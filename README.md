## :thinking: 什么是ClessS is All Around

《完蛋！我被克信包围了》是由弃车人二次开发的一款全新AIGC冒险游戏。游戏发生在一个被称作「5424」的直播间，在这里，被克勾选中的人将被授予「克信」，导引p话之力。你将扮演一位名为「弃车人」的神秘角色......同时，逐步发掘「薇克」的真相。
